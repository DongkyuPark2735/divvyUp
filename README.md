# "divvy Up(온라인 정산 시스템) 팀 프로젝트 포트폴리오"

> 팀 프로젝트

- 프로젝트 명 : divvy Up (온라인 정산 시스템) <br>
- 인 원 : 3 명 <br>
- 기 간 : 2022.07.28 ~ 2022.08.18 <br>
- DBMS : Oracle 11g Express Edition <br>
- 언어 : Java SE 8 Platform, HTML5, CSS, CSS3, JavaScript <br>
- 패턴 : MVC 패턴 <br>
- 오픈 Library : jQuery v3.4.1, apache POI 4.1.2 <br>
- 프로젝트 소개 : 그룹 단위 발생 소비의 정산 기능, 회원간 실시간 소통 게시판 기능,관리자의 기록 조회 기능 및 문서 다운로드 기능 <br>
